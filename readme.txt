Scroll to Top plugin allows the user to smoothly scroll at the top of the web page.

== Description ==

Scroll to Top plugin allows the user to smoothly scroll at the top of the web page with image or text of the button. this plugin gives you more options for the scroll to top button.

= Features =

Fully custom design.
Displays a button with an image or text.
Displays a button when a user scrolls down the page.
Change a button background color and text.
Set scroll position left or right.
Set button height and width.
Upload custom scroll icon.
Set the scroll offset.
Give the space from left/right and bottom.


== Installation ==

Easiest way:
1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Plugin Name screen to configure the plugin

Setup via FTP:

1. Download the plugin to your system and Unzip it.
2. Upload the "scroll-to-top" folder into your WordPress plugins directory.
3. Log in to your WordPress admin panel and go to the Plugins menu.
4. Then activate the plugin.
5. Go to Settings (Settings > Scroll to Top)
